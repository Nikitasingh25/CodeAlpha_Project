Project Overview ::

This project consists of two interactive dashboards developed using Power BI to analyze different types of data:
Employee Attrition Analysis Dashboard
Financial Analysis Dashboard
Both dashboards aim to transform raw data into meaningful insights using data cleaning, visualization, and analytical techniques.

Objectives ::

To perform data cleaning and transformation using Power Query
To create interactive dashboards using Power BI
To analyze employee behavior and financial performance
To extract insights for decision-making

Dataset Description

📁 1. Employee Attrition Dataset ::
Includes employee-related details such as:
Department
Gender
Job Role
Job Satisfaction
Attrition (Yes/No)

2. Financial Dataset :
Includes financial data such as:
State and Year
Revenue and Expenditure
Tax and other financial components
Sector-wise expenditure (Education, Health, etc.)

Data Cleaning & Transformation ::

The following steps were performed for both datasets:
Removed null and empty rows
Cleaned column names (handled spaces and special characters)
Converted data types (text, numeric, date)
Created calculated fields:
Profit = Revenue - Expenditure
Profit % = Profit / Revenue
Ensured data consistency for accurate analysis

This project demonstrates how data visualisation tools like power bi can be used to analyze both organisational and financial data effectively . This dashboard provide clear insights that support better decision making and performance analysis.
